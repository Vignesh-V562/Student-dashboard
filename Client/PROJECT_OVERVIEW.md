# Project Overview: Student Exam Dashboard

## Objective

The primary objective of this project is to create a modern, feature-rich, and user-friendly dashboard for students or educators. It serves as a central hub for managing and viewing academic information, with a strong focus on exam schedules and related activities. The application is a static prototype built with React, Vite, and styled with Tailwind CSS.

## Key Features

- **Three-Column Layout:** A responsive and resizable layout consisting of a main navigation sidebar, a central content area, and a right contextual sidebar.
- **Exam Management:** The core feature is the detailed display of exam schedules. It shows lists of exams with details like subject, time, status, and number of participants.
- **Comprehensive Navigation:** The left sidebar includes navigation for various academic sections such as 'Overview', 'Class Preparation', 'Attendance', 'Exams', 'Assignments', 'Schedule', and even a 'GPA Calculator'.
- **Interactive Calendar:** The right sidebar features a monthly calendar to help visualize exam dates and other events.
- **Upcoming Exams:** A dedicated panel highlights upcoming exams for quick reference.
- **Dark/Light Mode:** The entire UI supports toggling between dark and light themes for user comfort.
- **Dynamic UI Elements:** The interface is interactive with features like resizable sidebars and filters for the exam list.
- **Modern Tech Stack:** Built using modern front-end technologies including React, Vite for fast development, and Tailwind CSS for styling.
- **Rich Iconography:** Uses the `lucide-react` library for a clean and intuitive set of icons throughout the application.
